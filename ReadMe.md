# AI-Powered Learning Management System

A modern LMS platform built with FastAPI, Python, SQLite, and HTML/CSS.

## Features
- Student dashboard
- Teacher portal
- Admin management
- Assignment submission
- Attendance tracking
- Secure login system
- Role-based access

## Tech Stack
- Python
- FastAPI
- SQLite
- HTML/CSS
- GitHub
